# ArabOS-Kernel (شاهين) 🦅

## عن المشروع 🧠
المشروع عبارة عن نواة نظام تشغيل (Kernel) مخصصة لـ "نظام العرب"، تم بناؤها من الصفر لتكون الأساس البرمجي لإدارة العتاد والعمليات.

---

## بطاقة التعريف 📋

| الخاصية | التفاصيل |
| :--- | :--- |
| **اسم المشروع** | شاهين |
| **نوع المشروع** | نواة نظام تشغيل 64bit uefi (Kernel) |
| **هدف التصميمي** | نواة صغير (التصميم الدقيق) |
| **منشئ المشروع** | سيف حسين |

## إخلاء المسؤولية ⚠️
> **تنبيه:** لا يزال الإصدار الحالي تجريبياً (Alpha/Beta). لا يتحمل المطور أي مسؤولية عن أي سوء استخدام أو أضرار ناتجة عن تشغيل هذه النواة على أجهزة حقيقية. التجرية تقع بالكامل على مسؤولية المستخدم (يُنصح بالاختبار داخل بيئات التطوير المعزولة).
