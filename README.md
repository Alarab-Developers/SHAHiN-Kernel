# ArabOS-Kernel (شاهين) 🦅

## عن المشروع 🧠
المشروع عبارة عن نواة نظام تشغيل (Kernel) مخصصة لـ "نظام العرب"، تم بناؤها من الصفر لتكون الأساس البرمجي لإدارة العتاد والعمليات.
